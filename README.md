# 垃圾分类数据集

## 数据集说明
这是一个用于垃圾分类的图像数据集，包含40个细分类别，可归类为4个主要类别：可回收物、有害垃圾、厨余垃圾和其他垃圾。

## 数据集结构
```
datasets/
├── images/
│   ├── train/  # 训练集图片
│   └── val/    # 验证集图片
└── labels/     # 对应的标签文件
    ├── train/
    └── val/
```

## 类别信息
数据集包含以下类别：

### 可回收物（23类）
- 充电宝、包、化妆品瓶、玩具、塑料碗
- 塑料衣架、纸袋、插头电线、旧衣物、易拉罐
- 枕头、毛绒玩具、洗发水瓶、玻璃杯、鞋子
- 铁砧、纸板箱、调味品瓶、酒瓶、金属食品罐
- 锅、食用油桶、饮料瓶

### 有害垃圾（3类）
- 干电池
- 药膏
- 过期药品

### 厨余垃圾（8类）
- 剩饭剩菜、骨头、水果皮
- 纸浆、茶叶、蔬菜
- 蛋壳、鱼骨

### 其他垃圾（6类）
- 快餐盒、污损塑料
- 烟头、牙签
- 花盆、竹筷

## 使用方法
1. 配置文件 `data.yaml` 中已包含所有必要的配置信息
2. 数据集支持以下数据增强方式：
   - Mosaic 增强（比例：1.0）
   - MixUp 增强（比例：0.1）

## 标注格式
数据集使用 YOLO 格式标注，每个图片对应一个同名的 .txt 标注文件：
```
<class_id> <x_center> <y_center> <width> <height>
```

## 注意事项
1. 请确保按照 data.yaml 中指定的路径组织数据集
2. 训练前建议检查数据集的完整性
3. 标签ID与类别名称的对应关系请参考 data.yaml 文件
